
# Read me

## Requirements
1. ruby 3
2. node
3. postgresql server

## Setup
1. clone repository
2. run `bundle install`
3. configure `config/database.yml`
4. run `rails db:create db:migrate`

## Run project in development
1. run `rails s` to start server
2. run ` ./bin/webpack-dev-server` in second terminal
